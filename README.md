# Excel Shortcut Trainer for Data Analysts

An interactive web-based learning tool designed to help data analysts quickly memorize and practice essential Microsoft Excel keyboard shortcuts.

This project provides categorized shortcuts, interactive learning modes, quizzes, and a quick cheat sheet to improve productivity when working with Excel.

## 🌐 Live Demo

https://hanafidias.github.io/excel-shortcut-for-data-analyst

---

## 🚀 Features

### 📚 Shortcut Learning
Learn essential Excel keyboard shortcuts used in data analysis workflows.

Shortcuts are organized into multiple categories such as:

- Navigation shortcuts
- Editing shortcuts
- Formatting shortcuts
- Data management shortcuts
- Selection shortcuts

Each shortcut includes:
- Key combination
- Description
- Practical usage explanation

---

### 🔎 Search Shortcuts
Quickly find specific shortcuts using the built-in search feature.

---

### 📊 Progress Tracking
Track learning progress while exploring the shortcuts.

---

### 🎯 Quiz Mode
Test your knowledge of Excel shortcuts with an interactive quiz system.

Features include:
- Multiple-choice questions
- Instant feedback
- Final score summary

---

### 📋 Cheat Sheet
A quick reference page displaying all shortcuts in one place for fast review.
